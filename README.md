# 商品销售数据分析项目介绍
## 一、项目背景
本项目针对**2022年全年运动类商品销售数据**开展分析，核心目标是挖掘销售规律、识别核心商品与潜力品类，为企业库存优化、营销策略调整、定价策略优化提供数据支撑。

随着运动消费市场的增长，企业需要明确以下关键问题以提升经营效率：
1. 全年销售趋势如何，是否存在季节性波动？
2. 哪些小类商品贡献核心销售额，需重点关注？
3. 不同价格区间商品的市场接受度如何？
4. 核心畅销商品有哪些，是否需加强备货或推广？


## 二、数据概况
本次分析基于Excel文件中的两个核心数据表，通过数据合并与预处理形成最终分析数据集，具体信息如下：

### 2.1 原始数据表结构
| 数据表名称 | 核心字段 | 数据用途 |
|------------|----------|----------|
| 信息表     | 商品大类、商品小类、商品名称、商品编号、商品销售价 | 存储商品基础属性与定价信息 |
| 销售数据表 | 订单日期、订单编号、商品编号、订单数量 | 存储每笔订单的销售记录 |

### 2.2 数据预处理
1. **数据合并**：通过“商品编号”采用**右连接**，保留所有销售记录，最终形成8个字段的合并数据集
2. **衍生字段计算**：
   - 销售额 = 订单数量 × 商品销售价（核心业务指标）
   - 月份/月份_str：从订单日期提取年月，用于月度趋势分析
   - 季度：从订单日期提取季度，用于季节性分析
   - 价格区间：将商品销售价划分为6个区间（0-20元、21-40元等），用于价格维度分析

### 2.3 总体数据规模
- 合并后数据量：2040条销售记录
- 时间范围：2022年1月-2022年12月（全年数据）
- 核心统计指标：
  - 总销售额：5,280,308.00元
  - 总订单数量：2,040个（去重后）
  - 总销售商品数量：123,641件
  - 平均订单金额：2,588.39元


## 三、分析方法
本次分析采用“**数据聚合统计+多维度可视化**”的方法，分4个核心模块展开，具体如下：

### 3.1 总体月度销售趋势分析
- **统计方法**：按“月份”聚合，计算每月总销售额、总订单数量（件）、订单个数（去重）、平均订单金额
- **可视化工具**：2×2子图布局，包含：
  1. 折线图：展示每月总销售额趋势（含面积填充）
  2. 柱状图：展示每月总订单数量（件）变化
  3. 折线图：展示每月平均订单金额趋势
  4. 双轴图：对比销售额与订单数量的月度同步变化

### 3.2 小类商品销售趋势分析
- **统计方法**：按“月份+商品小类”聚合，筛选销售额前10的小类商品作为核心分析对象
- **可视化工具**：3×2子图布局，覆盖多维度视角：
  1. 堆叠面积图：展示小类商品销售额占比变化
  2. 热力图：呈现小类商品每月销售额热度分布
  3. 雷达图：对比关键月份（1/3/6/9/12月）小类商品表现
  4. 相关性热力图：分析前6小类商品订单数量的关联度
  5. 箱线图：展示前6小类商品销售额分布离散度
  6. 气泡图：关联小类商品的销售额、订单数量与商品种类数（气泡大小代表种类数）

### 3.3 补充维度分析（支撑策略调整）
针对业务核心痛点，新增5个关键维度分析：
1. **价格区间分析**：按价格区间聚合销售额、订单量，计算平均售价与平均订单金额
2. **商品大类分析**：按“运动上装/运动下装/运动配饰”聚合，计算销售额占比与平均单价
3. **商品排名分析**：分别按“销售额”和“订单数量”排序，筛选TOP10商品
4. **季节性分析**：按“季度”聚合，对比4个季度的销售表现
5. **价格弹性分析**：计算商品价格与销量的相关系数，判断价格对销量的影响

### 3.4 工具与技术
- 数据处理：Python（pandas、numpy）
- 可视化：matplotlib、seaborn（支持中文显示配置）
- 统计方法：相关性分析、数据分箱、聚合排序


## 四、核心结论与业务建议
### 4.1 总体销售趋势结论
1. **月度趋势**：7月销售额最高（490,090元），1月最低（344,878元），整体呈“上半年增长、下半年稳定”态势
2. **季节性特征**：销售额逐季递增，四季度最高（1,440,676元），一季度最低（1,151,564元），无明显淡季
3. **订单质量**：平均订单金额稳定在2,400-2,700元区间，11月最高（2,707.50元），订单客单价表现优质

### 4.2 商品结构结论
1. **大类贡献**：运动上装是核心品类，销售额占比62.16%（3,282,383元），平均单价61.41元；运动配饰占比最低（9.90%），但销量基数大（43,608件）
2. **小类TOP3**：外套（销售额最高）、背心、短袖，三者为核心利润贡献小类
3. **热门商品**：
   - 销售额TOP1：跑步宽松速干短裤（473,747元，运动下装）
   - 销量TOP1：半拉链立领长袖（5,760件，运动上装）
   - 运动配饰中，袜子、发带、手环销量进入TOP10，但单价低（8-16元）

### 4.3 价格策略结论
1. **价格区间表现**：61-80元区间销售额最高（2,040,494元），占比38.6%；41-60元次之（1,667,110元），两者合计贡献超70%销售额
2. **价格弹性**：商品价格与销量相关系数仅0.015，说明**价格对销量影响极小**，消费者更关注商品功能而非价格

### 4.4 业务优化建议
1. **库存策略**：
   - 重点备货运动上装（外套、长袖、背心）与运动下装（短裤、长裤），保障7-12月销售旺季供应
   - 运动配饰（袜子、发带）销量高但单价低，可采用“捆绑销售”提升客单价
2. **营销策略**：
   - 一季度（1-3月）推出“开春运动套装”促销，拉动淡季销售额
   - 针对TOP10商品（如跑步宽松速干短裤）打造爆款，加大流量倾斜
3. **定价策略**：
   - 优先推广61-80元、41-60元价格区间商品，符合市场接受度
   - 无需通过降价刺激销量，可通过提升商品品质维持现有定价
